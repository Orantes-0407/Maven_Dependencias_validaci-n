# Maven_Dependencias_validaci-n
La ofuscación dificulta considerablemente la lectura del código,
ya que los nombres de clases, métodos y variables son reemplazados
por identificadores cortos sin significado.

La estructura general del programa aún puede inferirse observando
el flujo de ejecución, pero la comprensión detallada requiere más
tiempo.

Sin embargo, el comportamiento del programa no cambia después de
la ofuscación, lo cual confirma que el proceso es funcional.
